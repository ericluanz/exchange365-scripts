# Exchange Online 365 - Scripts & Automação

Scripts e automações para gerenciamento do Exchange Online (Microsoft 365) utilizados no suporte e administração no meu dia a dia.

---

## 📂 Estrutura dos Scripts  

### 📧 Exchange Online  

* **Acesso a caixa postal.ps1**  
  Concede permissões de acesso a caixas de correio.  
  
* **Permitir enviar como outro e-mail.ps1**  
  Permite que um usuário envie e-mails em nome de outro ("Send As").  
  
* **Limite de envio e recebimento.ps1**  
  Configura limites de envio e recebimento de e-mails.  

* **Permissão de Calendário.ps1**  
  Define permissões de acesso a calendários.  

* **Alterar Idioma de contas e pastas.ps1**  
  Ajusta idioma padrão de caixas de correio e pastas.  

* **Alterar fuso horário.ps1**  
  Modifica o fuso horário de contas de usuário.  

---

### 🔗 Conexão e Utilidades

* **Conectar via PowerShell.ps1**  
  Script para autenticação e conexão com serviços Microsoft 365.

* **Exchange365 Scripts.ps1**  
  Script base com funções auxiliares para administração do Exchange.

---

### 📁 SharePoint Online

* **Listas do Sharepoint.ps1**  
  Criação e gerenciamento de listas no SharePoint.

* **PnP Online.ps1**  
  Conexão e operações utilizando PnP PowerShell.

---

## ⚙️ Pré-requisitos

Antes de executar os scripts, certifique-se de ter:  

* PowerShell 7  
* Módulos instalados:  

  * Exchange Online Management  
  * PnP PowerShell  
* Permissões administrativas adequadas no Microsoft 365
